Lista de comandos
   make keys	> cria a chave SSH e manda para maquinas
   make        	> compila + envia o binario
   make run    	> sobe a rede LAM e executa "make run 4" usa NP=4
   make seq    	> compila o sequencial
   make run-seq > roda o sequencial
   make clean  	> apaga os binarios locais

*dentro do make temos um campo para colocar o tempo do codigo sequencial, para entregar o calculo de eficiencia
